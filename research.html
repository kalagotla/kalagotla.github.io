Research

Research Philosophy

Understanding complex flow phenomena in aerospace systems requires a multidisciplinary approach that integrates theoretical frameworks, experimental techniques, and numerical simulations. I believe that the traditional Theory-Experiment-Numerical (T-E-N) methodology, while foundational, must evolve to embrace the data-centric paradigm enabled by modern computational capabilities.

My research philosophy centers on constructing feedback loops between data science and the T-E-N workflow, where physics-aware machine learning models serve as bridges that respect known physical constraints while uncovering patterns difficult to extract through conventional analysis.

Traditional T-E-N Methodology

Data-Centric DS-T-E-N Approach

Central to this approach is rigorous uncertainty quantification, particularly in experimental measurements where assumptions and measurement uncertainties can introduce significant biases. By developing comprehensive frameworks that account for empirical information and leveraging physics-informed neural networks, I aim to improve confidence in numerical codes while simultaneously providing deeper insights into the fundamental physics of flows.

This philosophy drives my work toward creating open-source tools that enable reproducible research and facilitate collaborative efforts between experimentalists and computational analysts.

Selected Projects

Bilateral Inertia-Correction Sparse Network (BICSNet)

Deep Learning • PIV Analysis • 2024-2025

A novel bilateral inertia-correction sparse network that achieves 76% error reduction in PIV data correction for oblique shock flows. This physics-aware neural network is specifically designed to correct for inertia bias effects in shock-dominated PIV measurements, trained on comprehensive synthetic datasets and demonstrating robust generalization across different Mach numbers.

Project Page GitHub AIAA SciTech 2025

Lagrangian Particle Tracking Library (lptlib)

Python Library • CFD • 2022-2025

A highly-parallelized one-way coupled Lagrangian particle tracking module with polydisperse simulation capabilities and Eulerian field conversion. This open-source library provides comprehensive particle dynamics simulation with customizable particle properties, modular design for diverse applications, and high-performance parallel computing support for large-scale simulations.

Project Page GitHub PyPI

Synthetic Image Generator for PIV (syPIV)

Computer Vision • Simulation • 2023-2024

A comprehensive synthetic image generator for emulating planar PIV data with advanced particle dynamics and parallelization support. This tool provides full planar PIV capability with realistic particle image generation, intelligent parallelization based on computational resources, and seamless integration with lptlib for end-to-end synthetic data generation pipeline.
