Research

Lagrangian Particle Tracking Library

A highly-parallelized one-way coupled lpt module.

Summary
  • Open source and available to install using pip.
  • Polydisperse simulation of advanced particle dynamics.
  • Converts Lagrangian data to Eulerian fields to reduce computational cost for further processing.
  • Modular design for diverse applications.
Synthetic Image Generator

A synthetic image generator to emulate planar PIV data.

Summary
  • Planar PIV capable
  • Parallelizing options based on size of the image.
  • Open source and easy installation.
  • API designed to accommodate future developments.
Bilateral Inertia-Correction Sparse Network (BICSNet)

A Deep Neural Network to correct for inertia bias effects in PIV data.

Summary
  • A novel architecture to correct for inertia bias effects in oblique shock PIV data.
  • Trained on synthetic data generated using the lptlib and Synthetic Image Generator.
  • Showcased an error reduction of 83% for in-distribution Mach numbers
