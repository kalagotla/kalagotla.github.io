Abstract

The uncertainty of the imaging phase and its interrogation phase in PIV contributes significantly to the total generated uncertainty. Synthetic image generation provides a powerful tool for validating numerical codes against PIV data while accounting for the uncertainty introduced during the imaging and processing phases.

syPIV is a comprehensive synthetic image generator for emulating planar PIV data, building upon the EUROPIV-2 Synthetic Image Generator (SIG). The code generalizes image generation for data obtained from Lagrangian Particle Tracking (LPT) codes, incorporating advanced particle dynamics and parallelization support. This tool enables researchers to evaluate PIV algorithms, understand PIV recording parameters before experimental setup, and validate numerical codes more effectively.

The current implementation has been applied to planar PIV setups and can be easily extended to stereo configurations. syPIV acts as a preliminary step in developing machine learning models that can handle uncertainties arising from the tracer, imaging, and analysis/interrogation phases of PIV.

Key Features

Core Capabilities

syPIV carefully models each aspect of the PIV process to produce realistic synthetic images:

  • Particle Distribution: Models both spatial particle distribution and physical particle diameter distributions, accounting for effects of coalescence, agglomeration, and breakup in supersonic flows
  • Interrogation Area: Accurately transforms particle locations from physical space to CCD sensor coordinates, accounting for laser sheet thickness and optical geometry
  • Laser Properties: Incorporates monochromatic laser sheet properties including wavelength, pulse energy, pulse width, and intensity profiles (Gaussian or top-hat)
  • Optics System: Models particle image intensity using 2D Gaussian distributions, accounting for particle diameter, laser sheet intensity, and pixel fill ratios

Integration with lptlib

syPIV seamlessly integrates with the Lagrangian Particle Tracking Library (lptlib) to provide an end-to-end synthetic data generation pipeline. This integration allows for:

  • Direct use of particle trajectory data from LPT simulations
  • Incorporation of particle dynamics history (PDH) effects
  • Realistic modeling of particle lag in acceleration regions, particularly important for supersonic flows

Parallelization Support

The code includes intelligent parallelization based on available computational resources, enabling efficient generation of large synthetic image datasets required for machine learning applications and comprehensive PIV algorithm evaluation.

Validation Results

syPIV has been validated against experimental PIV data for oblique shock flows, demonstrating the importance of incorporating particle dynamics history (PDH) effects in synthetic image generation.

Oblique Shock Analysis

Synthetic images were generated for the 20° deflection case of Williams et al. (2015), incorporating PDH effects through integration with the LPT code. The images were processed using openPIV with a 48×48 interrogation window size, matching the experimental PIV analysis parameters.

When overlaying synthetic images generated from analytical data and PDH-informed data, the particle lag across the oblique shock becomes evident. The color-temperature-corrected overlapped images highlight this particle lag as streaks, which are clearer at the shock interface, demonstrating the effect of particle inertia in the PDH-informed data.

Velocity Field Comparison

The processed vector fields for both analytical and PDH-informed data demonstrate the qualitative impact of particle presence on optical velocimetry data. The overlapped velocity vectors highlight the differences between traditional synthetic PIV (assuming particles carry local flow velocity) and PDH-informed synthetic PIV.

Quantitative Validation

Comparison of shock normal velocity lag data reveals significant improvements when using PDH-informed synthetic images:

Average Error (RMS, 0-20mm)

  • Traditional syPIV vs. experiment: 15.8%
  • LPT code vs. experiment: 0.6%
  • PDH-informed syPIV vs. experiment: 0.2%

Maximum Error

  • Traditional syPIV vs. experiment: 86%
  • LPT code vs. experiment: 2%
  • PDH-informed syPIV vs. experiment: 1.7%

Traditional syPIV data assume that particles carry the local flow velocity in the first image, leading to substantial errors in velocity lag compared to both the LPT code and PDH-informed data. The difference in error between PDH-informed data and the numerical code suggests that image analysis did not produce significant uncertainty in this case. The maximum error values demonstrate that PDH is locally substantial in acceleration regions when studying flows with tracers.
