Abstract

This work presents the application of BICSNet (Bilateral Inertia-Correction Sparse Neural Network) to experimental shock-shock interaction particle image velocimetry (PIV) data obtained from the Polysonic Wind Tunnel at Florida State University. The study demonstrates the network's capability to correct particle inertia bias in real-world supersonic flow measurements, validating the model against computational fluid dynamics (CFD) simulations and experimental data.

The experimental campaign investigated shock-shock interactions generated by wedge shock generators at Mach 2, with BICSNet applied to correct for particle dynamics history (PDH) effects in the PIV measurements. Results show significant improvements in velocity field accuracy, with the corrected data showing better agreement with CFD predictions. However, the network fails to correct in the shock interaction region, which was not included in the training data.

Methodology

Key Points

  • BICSNet is a bilateral CNN architecture enhanced with U-Net components
  • Operates directly on raw PIV image pairs with physics-aware inputs
  • Trained on 76,800 synthetic image sets (Mach 1.5-7.6)
  • Requires careful preprocessing to match synthetic training data characteristics

BICSNet is a bilateral convolutional neural network (CNN) architecture enhanced with U-Net components, specifically designed to correct particle inertia bias in PIV measurements. The network operates directly on raw PIV image pairs, extracting localized particle features through convolutional layers while preserving fine-scale spatial information critical for resolving shock structures.

The architecture incorporates physics-aware inputs, including freestream Mach and Reynolds numbers computed from particle properties. These scalar inputs are integrated into the deep decoder layers, providing essential physical context that allows the network to adapt the correction process to specific flow conditions. Unlike physics-informed neural networks (PINNs), this approach maintains computational efficiency while learning image features and adhering to underlying physics.

The model was trained on approximately 76,800 synthetic image sets representing diverse oblique shock conditions, including variations in Mach numbers (1.5-7.6), deflection angles, shock strengths, particle properties, and magnifications. However, applying the trained network to experimental PIV data requires careful preprocessing to match the synthetic training data characteristics.

Experimental Data Preprocessing

The experimental PIV dataset comprises 365 particle image pairs captured by a 2048×2048 pixel camera. Since BICSNet was trained on synthetic data with oblique shocks aligned to the Cartesian grid, the experimental shock-interaction data must be preprocessed to enable meaningful feature extraction and bias correction.

1 Rotation and Cropping

The first preprocessing step involves rotating and cropping the incident shock region to align with the Cartesian grid, as illustrated in the figure. This alignment is crucial because the model was trained on synthetic data where shocks are oriented vertically or horizontally relative to the image axes.

The rotation ensures that the shock front is properly oriented for the network to recognize and process the flow features effectively. The cropped region isolates the shock interaction zone of interest, removing extraneous flow features that could confuse the model.

2 Patch Extraction

To process the full shock interaction data, each 2048×2048 pixel image is cropped into 256×256 pixel patches with 50% overlap. This patching strategy produces 225 cropped images per PIV frame, ensuring comprehensive coverage of the flow field while maintaining the input size required by BICSNet (256×256×3).

The 50% overlap ensures that no flow features are missed at patch boundaries and provides redundancy that helps improve the robustness of the correction process. Each patch is processed independently through BICSNet, and the results are later reassembled to reconstruct the full flow field.

Total dataset: This procedure generated 94,095 image pairs from the 365 experimental frames, providing a substantial dataset for validation and analysis.

3 Intensity Distribution Matching

A critical challenge in applying BICSNet to experimental data is matching the intensity distribution characteristics of the synthetic training data. Experimental PIV images exhibit significantly different intensity profiles compared to synthetic images, with approximately 80% of pixels at zero intensity (black levels) versus 25% in synthetic data.

Preprocessing techniques:

  • Gaussian Blur: Applied with a 3×3 kernel (matching particle size in pixels) to smooth intensity distributions and reduce noise
  • Random Noise: Added using a normal distribution with standard deviation of 5.0 to improve intensity count smoothness
  • Exposure Factor: Multiplied by image intensity (0.8 for incident shock, 0.95 for transmitted shock) to reduce particle streaking effects

These preprocessing steps effectively reduce zero-intensity pixel counts from approximately 75% to 30%, bringing the experimental data distribution closer to the synthetic training data.

4 BICSNet Processing

Once preprocessed, the 256×256 pixel image patches are passed through the trained BICSNet model along with scalar inputs (Mach number = 2.0, particle Reynolds number = 427). The network processes each patch independently, applying physics-aware corrections to reduce particle inertia bias. The corrected patches are then reassembled to reconstruct the full velocity field, which can be further analyzed using standard PIV post-processing tools like OpenPIV.

This preprocessing pipeline ensures that experimental PIV data is compatible with the BICSNet architecture while preserving the essential flow physics, enabling accurate correction of particle inertia bias in real-world supersonic flow measurements.

Computational Fluid Dynamics

CFD simulations were conducted using ANSYS Fluent 2021 R2 to provide validation data for the experimental PIV measurements. The computational domain models the shock-shock interaction generated by 10° wedge shock generators at Mach 2, matching the experimental configuration.

An adaptive mesh refinement strategy was employed to capture the shock interaction region accurately. The initial grid contained 33,097 cells with a maximum cell thickness of 0.1 mm in the shock interaction region. The refinement process utilized gradient adaptation with a factor of 0.5, splitting cells up to 8 levels of refinement.

This process yielded a final grid resolution of approximately 10⁻⁵ m in the shock cells, enabling detailed resolution of the complex flow structures. The adaptive refinement ensures computational efficiency while maintaining high accuracy in critical regions.

The Reynolds-averaged Navier-Stokes (RANS) equations were solved using the k-ω turbulence model, chosen for its robustness in capturing regions with adverse pressure gradients typical of shock interactions. Boundary conditions were set as follows: pressure inlet with gauge total pressure of 280,000 Pa, supersonic/initial gauge pressure of 35,785.27 Pa, and total temperature of 291.25 K; pressure outlet with gauge pressure of 35,785.27 Pa; far-field pressure conditions on domain boundaries; and no-slip walls on the wedges.

The converged CFD solution provides velocity contours normalized by the free-stream velocity of 510 m/s, offering a high-resolution reference for comparison with experimental PIV data and BICSNet-corrected results. The CFD data exhibits finer resolution due to the higher grid refinement, allowing for a more detailed representation of the flow structures compared to the experimental measurements.

Experimental Setup

Experimental data for validating BICSNet were obtained from high-speed tests conducted in the Polysonic Wind Tunnel (PSWT) at Florida State University. This blowdown-type tunnel operates over a Mach number range of 0.2 to 5 and features a 0.305 m × 0.305 m solid-wall test section with optical access from four sides.

For the current Mach 2 experiments, a fixed nozzle block was used, and the tunnel was supplied with 150 m³ of air at 3.5 MPa, enabling run times of approximately 60 seconds. The flow was unheated, with near-ambient stagnation temperatures monitored by a Resistance Temperature Detector (RTD), and pressures were recorded using Baratron transducers.

The test campaign investigated shock-shock interactions generated by wedge shock generators with leading-edge deflections of 5°, 10°, and 15°. Each wedge had a 0.14 m span to balance 2D flow development with minimal blockage. The generators were mounted on diamond cross-section struts from the ceiling and floor. The present study focuses on data from the 10°/10° wedge configuration at Mach 2.

PIV measurements were performed in a vertical streamwise plane between the wedges, with a laser sheet introduced through the top of the test section. An Nd:YAG laser (EVG-200) operating at 7.23 Hz with 200 mJ/pulse and an inter-pulse delay of 0.8 μs was used to illuminate the flow field. Two LaVision Imager Pro X cameras captured images from the side window: one wide-angle camera for the overall flow and one zoomed-in camera to capture the shock-shock interaction zone.

Seeding for the PIV system was generated using a Wright nebulizer housed in a pressure vessel. The nebulizer used Rosco Fog Fluid, a glycol-based commercial seeding solution. Compressed air from the tunnel supply system was regulated to 0.55 MPa to drive the nebulizer. The air-seed mixture was introduced into the tunnel's stagnation chamber via a vertical seed pipe with five upstream-facing holes to ensure proper mixing. Particle size distributions were analyzed to characterize the seeding properties, with distributions showing typical characteristics of nebulizer-generated particles.

Results

The application of BICSNet to the experimental PIV data demonstrates significant improvements in velocity field accuracy, particularly in regions affected by particle inertia bias. Comparison of CFD velocity contours with PIV measurements reveals the impact of particle dynamics history (PDH) effects, where particles lag behind the fluid motion as they traverse through shock structures.

The corrected velocity fields show improved agreement with CFD predictions, especially in the shock interaction region where particle inertia effects are most pronounced. The velocity profiles demonstrate how BICSNet successfully bridges the gap between raw PIV measurements and CFD validation data.

For the incident oblique shock, BICSNet processing achieved a 53.75% reduction in inertia bias when compared to CFD validation data. The normalized velocity profiles at different y-locations (y = 0.5 mm and y = 5 mm) clearly show that BICSNet-corrected data closely matches the CFD predictions, while raw PIV data exhibits the characteristic smearing and lag due to particle inertia effects.

The results demonstrate BICSNet's ability to generalize from synthetic training data to real-world experimental measurements, validating the network's effectiveness in correcting particle-induced uncertainties in supersonic PIV data. The corrected data provides a more accurate representation of the true flow field, bridging the gap between experimental and numerical analyses.

Shock-Shock Interaction Results

The entire shock interaction dataset was processed using BICSNet, with the raw PIV images segmented into pre-interaction and post-interaction zones. For the pre-interaction region, model inputs were a freestream Mach number of 2.0 and particle Reynolds number of 427. For the post-interaction region, the Mach number was set to 1.6 (obtained from oblique shock relations) with a particle Reynolds number of 402.

(a) PIV Data

Raw PIV measurements show smeared shock fronts due to particle inertia bias, with gradual velocity transitions across the shock waves.

(b) BICSNet Data

BICSNet-corrected data exhibits significantly sharper shock gradients and better-aligned shock locations compared to the raw PIV data.

(c) CFD Data

CFD simulation provides the reference validation data, showing the expected sharp, well-resolved shock structures.

Normalized velocity contours for shock-shock interaction. The BICSNet-corrected contours closely match the CFD predictions, particularly in the pre-interaction and post-interaction regions. In the interaction zone where two shock fronts intersect, some residual bias remains, but overall improvement is substantial with error reductions of 54.26% at the first shock and 58.41% at the second shock.

Cross-correlation using OpenPIV was implemented to extract velocity fields from the corrected images, which were then stitched by averaging overlapping regions to reconstruct the complete shock interaction domain. In the reconstructed BICSNet data, shock locations are better aligned and exhibit sharper gradients compared to the original PIV fields. The quantitative improvements demonstrate the model's interpolation capabilities, as the Mach 2.0 case was not included in the training set, yet BICSNet achieved significant error reduction.
